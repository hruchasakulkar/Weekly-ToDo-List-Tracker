# 📝 Weekly To-Do List Tracker

A command-line to-do list application built in Python that helps users organize their weekly tasks by day and priority. The project uses nested dictionaries to store multiple tasks for each day and displays them in both chronological and priority-based views.

## Features

- Add multiple tasks for each day of the week.
- Assign priorities (High, Medium, Low) to every task.
- Input validation for priorities and user responses.
- View all tasks organized by day.
- Display tasks grouped by priority for each day.

## Concepts Practiced

- Nested Dictionaries
- Dictionary Traversal
- Lists
- Nested Loops
- User Input Validation
- Conditional Statements
- Data Organization

## Technologies Used

- Python

## How to Run

1. Ensure Python is installed.
2. Run the program:
   ```bash
   python to_do_list_tracker.py
   ```
3. Follow the prompts to create and manage your weekly task planner.
